## What is HERITRACE?

HERITRACE facilitates non-technical domain experts in enriching and editing metadata in a semantically robust manner. It is developed with a focus on user-friendliness, provenance management, change tracking, customization, and integration with heterogeneous data sources.

## Key features

- **User-Friendly Interface**: Non-technical domain experts can easily enrich and edit metadata
- **Provenance Management**: Complete tracking of who made changes, when, what modifications were made, and data sources used
- **Customizable Schemas**: SHACL-based data models and YAML display rules
- **RDF Data Integration**: Seamless import of existing RDF data from triplestores without modification
- **Secure Authentication**: ORCID-based access control ensuring only authorized personnel can make changes
- **Version Control**: Full versioning and rollback capabilities through Time Machine functionality
